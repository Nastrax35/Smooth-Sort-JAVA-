# Cеместровая: Алгоритм Smoothsort


## Состав программы
Программа выполнена на языке **Java** и включает в себя:
*   **Генератор данных:** Автоматическое создание 100 наборов случайных чисел (от 100 до 10 000 элементов) и сохранение их в файлы `.txt`.
*   **Модуль тестирования:** Поэтапное считывание данных и замер показателей производительности (наносекунды и количество итераций).
*   **Алгоритм Smoothsort:** Реализация на основе чисел Леонардо с адаптивным механизмом упорядочивания.

## Результаты измерений
В ходе работы были получены следующие данные:
1.  **Временная сложность:** Зависимость времени выполнения от размера входных данных (подтверждено соответствие тренду $O(n \log n)$).
2.  **Вычислительная трудоемкость:** График количества итераций, сопоставленный с теоретической кривой сложности.

## Технические требования
*   Java Development Kit (JDK) 8 или выше.
*   Среда исполнения (IntelliJ IDEA, Eclipse или терминал).

## Запуск
1. Скомпилируйте файл: `javac SmoothSortLab.java`
2. Запустите программу: `java SmoothSortLab`
3. Результаты измерений будут выведены в консоль в формате CSV для последующего анализа в Excel.
