# Реализация алгоритма Smoothsort (Плавная сортировка)

## Файлы проекта
* **SmoothSortLab.java** — основной класс, содержащий:
    * Реализацию алгоритма Smoothsort.
    * Программный генератор тестовых наборов данных.
    * Модуль чтения данных из файлов.
    * Систему замера метрик (время и итерации).
* **data/** — папка со сгенерированными входными данными (`set_100.txt`, `set_200.txt` и т.д.).
* **results.csv** — результаты запусков в формате, пригодном для импорта в Excel/Google Sheets.

## Формат входных данных (data/*.txt)
Каждый файл содержит последовательность целых чисел (Integer), разделенных переносом строки.
Данные генерируются программно и считываются непосредственно перед запуском алгоритма.

## Выходные метрики
При запуске в консоль выводятся данные в формате CSV со следующими колонками:
* **Size** — количество элементов в массиве (от 100 до 10 000).
* **Iterations** — количество итераций (внутренний счетчик сравнений и перестановок).
* **Time_ns** — время работы алгоритма в наносекундах (исключая время считывания из файла).

## Что измеряется
* **iterations** — сумма операций просеивания (sift) и перестроения кучи.
* **time_ns** — чистое время работы метода `smoothSort`, замеряемое с помощью `System.nanoTime()`.

## Запуск
1. **Скомпилировать:** 
   ```bash
   javac SmoothSortLab.java
   ```
2. **Запустить:** 
   ```bash
   java SmoothSortLab
   ```

## Особенности реализации
Алгоритм является адаптивным: при частичной упорядоченности входных данных сложность стремится к **O(n)**. В среднем и худшем случаях сложность составляет **O(n log n)**. Реализация выполнена "на месте" (in-place) без использования дополнительной памяти.
